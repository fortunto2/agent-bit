## Suggested Patches

### Patch 1: solo-dev.sh — Pre-check plan completion

**What:** Auto-archive 100%-done plans before entering build stage
**Why:** Stale plan cycling wasted iterations (Pattern 1) — unfixed from prior retro

```diff
# Before entering build stage, check if plan is already complete
+ if [ -n "$active_plan" ]; then
+   total=$(grep -c '\- \[' "$active_plan/plan.md" 2>/dev/null || echo 0)
+   done=$(grep -c '\- \[x\]' "$active_plan/plan.md" 2>/dev/null || echo 0)
+   if [ "$total" -gt 0 ] && [ "$total" -eq "$done" ]; then
+     log "QUEUE" "Plan $active_plan is 100% done — auto-archiving"
+     mv "$active_plan" "$project_root/docs/plan-done/"
+     continue
+   fi
+ fi
```
