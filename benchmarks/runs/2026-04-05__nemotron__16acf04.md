## Key Finding

SYSTEM_PROMPT_EXPLICIT content is NOT bloat — it's essential optimization for Nemotron's weak reasoning:
- DECISION FRAMEWORK paragraph: defines CRM scope, prevents false DENIED/CLARIFICATION
- Verbose decision tree: each step needs detailed context for Nemotron to evaluate correctly
- Expanded KEY section: operational hints (email=CRM, counting via search, Salesforce=UNSUPPORTED)
- INBOX PROCESSING: guides per-message evaluation strategy

Only safe change: PLANNING_PROMPT duplicate pattern removal (contact-ambiguity + multi-inbox).
